Emacs configuration entry point

* Initialize package sources and use-package
#+begin_src emacs-lisp
  ;; Initialize package sources
  (require 'package)
  (setq package-archives '(("melpa" . "https://melpa.org/packages/")
                           ("org" . "https://orgmode.org/elpa/")
                           ("elpa" . "https://elpa.gnu.org/packages/")
                           ("nongnu" . "https://elpa.nongnu.org/nongnu/")))
  (package-initialize)
  (unless package-archive-contents
    (package-refresh-contents))

  ;; Initialize use-package on non-Linux platforms
  (unless (package-installed-p 'use-package)
    (package-install 'use-package))
  (require 'use-package)
  (setq use-package-always-ensure t)
#+end_src

* Load files

** Add init-files directory to load-path
Add the `./config/` directory to the load path since.
#+begin_src emacs-lisp
  (add-to-list 'load-path (expand-file-name "config/" user-emacs-directory))
#+end_src

** Require individual files
Functions must come first since other modules may reference these helpers.
#+begin_src emacs-lisp
  (require 'config-functions)
  (require 'config-ui)
  (require 'config-editor)
  (require 'config-packages)
  (require 'config-org-mode)
  (require 'config-language-modes)
#+end_src

** Require work config
Conditionally load your work specific config file. The work config file just needs to be named, `work.el`, needs to be in the `init-files` directory, and needs to (provide 'config-work) in the file.
#+begin_src emacs-lisp
  (let ((work-file (expand-file-name "init-files/work.el" user-emacs-directory)))
    (when (file-exists-p work-file)
      (require 'config-work)))
#+end_src
